Justification

Design Choices

The layout is simple and clean to match the elegant theme of weddings. I organized the website into clear sections for bridal dresses, guest dresses, and jewellery, making it easy for users to find what they want. There are plenty of blank spaces throughout the site, including ample use of headings and images, all of which work together to create an environment that is very user friendly.

Color Choices

Soft colors were used throughout the web-site (such as white, gold, and blush pink), due to the romantic and luxurious quality of weddings. Soft colors used on this web-site allow the dresses and jewelry to be clearly visible, while keeping distractions to a minimum for the user.

HTML Structure Choices

Semantic HTML elements (header, nav, main, and section) have been used to organize the coding on this site. Organizing coding will enhance navigation for both humans and machines (search engine spiders, screen readers), and assist in identifying content on the web-site.

CSS Styling Choices

An external CSS file was utilized to isolate styling from the HTML, which made it easier to update the styling. Consistent styling (fonts, colors, spacing) using CSS has assisted in creating a visually appealing and professional look on the web-site.

JavaScript Choices

JavaScript has been used for simple interactive functions (form validation), to ensure that users input valid data. The use of JavaScript has improved the user experience, and eliminated errors resulting from user error when submitting forms.

Bootstrap Choices

Bootstraps Grid System was utilized to create a neat layout on the web-site, and to ensure that the web-site remains responsive on mobile devices, tablets, and laptops. Additionally, Bootstraps Grid System provides pre-built classes for layout and spacing, which assists in speeding up the development process.

Maintainability

Clean HTML, external CSS, Bootstrap, and minimal use of JavaScript creates a web-site that is easy to maintain. Adding new dresses, images, or pages to the web-site does not require updating the entire web-site's structure; therefore, maintaining the web-site in an organized fashion is much easier.
